# Chaitanya Surabattuni — Portfolio Website 🚀

Personal portfolio website for **Chaitanya Surabattuni**, AI Engineer.

Built with React, TypeScript, GSAP, Three.js, and WebGL.

---

## ⚙️ Tech Stack

React • TypeScript • GSAP • Three.js • WebGL • HTML • CSS • JavaScript

---

## 🛠️ Setup

```bash
npm install
npm run dev
```

> ⚠️ Note: This project uses GSAP Club trial plugins which cannot be used for production hosting.
> For official GSAP Club plugins, see: https://gsap.com/docs/v3/Installation/
